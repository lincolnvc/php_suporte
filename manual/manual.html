Carregar processo


O primeiro passo é fazer o upload do conteúdo dentro da pasta "sistema-ticket" (NOTA IMPORTANTE: Publique o que está dentro da pasta Script, não a pasta de script) para a sua pasta public_html ou em sua pasta raiz padrão FTP.

A segunda coisa é fazer o upload para o seu banco de dados MySQL usando a função de importação disponível no MySQL o arquivo ticket.sql da pasta chamada "data / sql"




processo de instalação

Com um editor de texto (in: Notepad ++), abra o seguinte arquivo localizado dentro da pasta 'script':

PATH: script do nome / app / config / database.php

Agora, substituir o que está dentro da matriz com suas informações, você deve ter algo parecido com isto:




        'mysql' => array(

            'driver'    => 'mysql',

            'host'      => '',

            'database'  => '',

            'username'  => '',

            'password'  => '',

            'charset'   => 'utf8',

            'collation' => 'utf8_unicode_ci',

            'prefix'    => '',

        )
